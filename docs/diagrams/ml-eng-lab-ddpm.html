ml-eng-lab — DDPM Diffusion (Forward + Reverse)

Forward q(x_t|x_0) corrupts a clean image into pure noise; a trained DiffusionMLP denoiser ε_θ inverts the trajectory via p_θ(x_{t-1}|x_t).

Forward (analytic)

  • q(x_t|x_0) adds Gaussian noise on a fixed β schedule.
  • Closed-form — x_t sampled without ever running a model.
  • x_T ≈ N(0, I) destroys all structure by step T.

Denoiser ε_θ

  • Collection uses `DiffusionMLP`, not a U-Net.
  • Takes (x_t, sinusoidal-t), predicts the noise ε̂.
  • Trained by simple MSE: ‖ε − ε_θ(x_t, t)‖².

Reverse (learned)

  • Sample x_T ~ N(0, I), iterate t = T → 1.
  • Each step uses ε_θ to compute μ_θ, then adds σ_t·z.
  • End state x_0 is a freshly generated digit/garment.
