ml-eng-lab — Autoencoder

Hourglass feed-forward network: input → encoder → 2-D bottleneck latent → decoder → reconstruction, trained by reconstruction loss.

Hourglass

  • 4-D Iris input narrowed to a 2-D latent, then widened back to 4-D.
  • Bottleneck is the only information channel — it forces compression.

How it is built here

  • One FeedFwdNN(input_dim == output_dim) instance trains the whole map.
  • Unsupervised: the only signal is reconstruction loss ‖x − x̂‖².

What the latent buys you

  • 2-D embedding for linear-probe visualisation.
  • Feature space for KMeans clustering; nonlinear where PCA is linear.
