ml-eng-lab — Graph Neural Network Message-Passing

Center node v aggregates neighbor messages, updates its embedding, and stacks two layers to widen its receptive field.

One layer, one recipe

  • Neighbors u1, u2, u3 send messages into AGGREGATE.
  • AGGREGATE pools them (mean / max / sum) into a_v.
  • UPDATE fuses a_v with self feature h_v^(ℓ) → h_v^(ℓ+1).

Stacking layers

  • Layer 0 = raw features; Layers 1–2 each run one MP step.
  • Each hop widens v's receptive field by one neighbor ring.
  • Two stacked layers ⇒ 2-hop structural awareness.

Three collection variants

  • GraphConv — spectral mean aggregation.
  • GraphSAGE — sampled, inductive aggregator.
  • GAT — attention-weighted neighbor sum.
