ml-eng-lab — Feed-Forward MLP Architecture

The collection's nnx.FeedFwdNN: input → hidden layer(s) (Linear → LeakyReLU → dropout) → linear logits → softmax cross-entropy. Forward pass left→right; backpropagation gradient flow right→left.

Topology

  • dims = [in, h₁, h₂, …, out]; one nn.Linear per adjacent pair.
  • Iris sweep: [4] → [4,8,3] → [4,16,8,3]; MNIST: 784 → … → 10.
  • Input is feature data; hidden + output layers are the model.

Forward pass

  • Hidden: zₗ = Wₗaₗ₋₁ + bₗ, aₗ = σ(zₗ) (LeakyReLU) + dropout.
  • Last layer is linear — returns logits; no σ before the loss.
  • Softmax is folded into nn.CrossEntropyLoss (expects raw logits).

Backpropagation

  • PyTorch autograd flows ∂L/∂Wₗ, ∂L/∂bₗ right → left.
  • Adam update: Wₗ ← Wₗ − η · ∂L/∂Wₗ.
  • Dropout on in .train(), off in .eval() — same forward graph.
