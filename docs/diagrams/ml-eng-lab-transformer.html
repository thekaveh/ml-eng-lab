ml-eng-lab — Decoder-Only Transformer Block

Forward pass of an autoregressive (decoder-only) transformer: token ids → embedding + RoPE → N× masked-attention/FFN blocks → LM head → next-token logits.

Input pipeline

  • Token ids → learned embedding lookup → [B, T, d_model].
  • RoPE injects position by rotating Q, K inside attention (no learned params).
  • Stream enters the N× block stack autoregressively.

Transformer block ×N

  • Masked MHSA: softmax(QKᵀ/√d)·V under a lower-triangular causal mask.
  • Residual + LayerNorm wraps both attention and the feed-forward network.
  • FFN: Linear(d, 4d) → GELU → Linear(4d, d); repeated N times.

Output head

  • Final LayerNorm → linear LM head projects hidden state to vocab logits.
  • Softmax yields the next-token distribution; sample id_{t+1}.
  • Sampled token appends to the sequence; repeat for generation.
